# 🧠 LLM Explorer

Bem-vindo ao meu laboratório de Inteligência Artificial Generativa. Este repositório é dedicado ao estudo prático de Large Language Models (LLMs), orquestração com **LangChain** e fluxos de agentes complexos com **LangGraph**.

---

## 🚀 Objetivo
Documentar minha evolução técnica na construção de aplicações inteligentes, saindo de simples chamadas de API para sistemas de agentes autônomos e RAG (Retrieval-Augmented Generation) avançado.

## 🛠️ Tecnologias Principais
* **Linguagem:** Python 3.10+
* **Orquestração:** LangChain & LangGraph
* **Modelo:** Google (Gemini)
